About Exactly

Exactly is a small set of free, no-nonsense web tools for one frustrating job: getting a file to the exact size or format something requires. No sign-up, no watermark, and nothing is ever uploaded — every tool runs entirely inside your own browser.

The tools

  • Compress to size — shrink any JPG, PNG or WebP image to a precise kilobyte target, like "under 50 KB", using a search that lands at or just below the number you ask for.
  • Passport & visa photo — check a photo against official requirements for several countries, and crop it to the correct dimensions, pixel size and file size if it doesn't yet pass.
  • Convert data files — convert between JSON, YAML, XML and CSV in any direction, for developers and anyone working with config files.

Why "nothing is uploaded" matters

Most online file tools send your file to a server to process it. Exactly doesn't. Everything happens locally on your device using standard browser technology, so your photos, documents and config files never leave your computer or phone. It's faster, and it's private by design.

A note on accuracy

The passport and visa specifications are gathered from official and authoritative sources and are kept as current as we can, but government requirements change and can vary by consulate or portal. Always confirm the latest rules on the official government website before you submit anything. Exactly handles the measurable parts — size, crop, format and file size — and never guarantees that an application will be accepted.
